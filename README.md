## Getting Started


First, install all dependencies:

```bash
npm install
# or
yarn
```

Second, install and run json server:

```bash
npm install -g json-server
yarn server
```

third, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [Run expo](https://docs.expo.io/)
